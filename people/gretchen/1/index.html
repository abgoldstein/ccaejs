Hello, JavaScript.

Let's talk about the history later. Here is what you practically need to know right now:

  • JavaScript is the language of web browsers. Using it, you can manipulate elements contained within the Document Object Model
  • JavaScript was created in 10 days in 1995 by Brendan Eich at Netscape; it shows. However, the community has adopted the good parts and matured them
  • JavaScript is a loosely typed language. White space does not matter. Comments are done C-style (// to comment a line, /* */ for longer comments)
  • If you are comfortable with a C-like language, the basic syntax will feel immediately familiar

Deep down, JavaScript has more in common with Lisp and Scheme than with Java. It is Lisp in C's clothing. This makes JavaScript a remarkably powerful language.

Douglas Crockford, Javascript: The Good Parts

OSX Basics - Hotkeys

Cmd here mean the Command key. It may be labeled on your keyboard as just the Apple logo. Or if you are using a keyboard intended for Windows, the Command key is the Windows key.

  • Copy - Cmd + C
  • Paste - Cmd + V
  • New Directory - Cmd + Shift + N
  • Switch Applications - Cmd + Tab
  • Switch Between Application Windows - Cmd + ~
  • Opening Spotlight - Cmd + Space

Using the Terminal

The terminal can be opened by hitting Cmd + Space to open Spotlight and then typing Terminal. The first result will be Terminal.app, so you can hit enter to open the application. These are some basic commands:

  • List files - ls
  • Change directories - cd [destination]
  • Show current directory - pwd
  • Copy - cp [source] [destination]
  • Move - mv [source] [destination]
  • Delete - rm [file]
  • Delete directory - rm -r [directory]
  • Make directory - mkdir [name]

Using the Terminal - A quick exercise

Let's put those commands to work. We're going to make a directory to save your projects

  1. Navigate to your home directory - cd ~
  2. Make a new directory for code. Let's call it "workspace" - mkdir workspace
  3. Navigate into that directory - cd workspace
To make your life easier, try using tab as you type file names. The terminal will auto-complete filenames and commands to the best of its ability. Try it! Navigate back to your home directory, type cd w and then hit tab. It's a sanity saver!

Variables - Types

JavaScript is a loosely typed language. You do have these types:

  • Strings - Stuff inside of quotes "" or single quotes ''
  • Numbers - Any number. 1, 2, 3.14, and 10e9 are all valid
  • Booleans - true or false
  • Arrays - Lists []
  • Objects - {}

Variables - var

The var keyword is special and pertains to scope. We'll address it later. For now, rules of thumb:

  • The first time you define a variable, write it as var x = 2
  • When redefining a variable that has been made before, write it as x = "I'm a string now"
Note: Variables are dynamic in JavaScript. In the above example, x started as a number and then we redefined it as a String
